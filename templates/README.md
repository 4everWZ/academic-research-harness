# Paper Workspace

This directory is a flat research-writing workspace.

Core files:

- `venue_profile.md`: target venue/outlet writing tendencies and format assumptions.
- `paper_index.md`: literature metadata, role classification, and evidence quality.
- `references.bib`: BibTeX records for indexed references.
- `claims.md`: claim ledger and support status.
- `idea_log.md`: literature-grounded idea refinement notes.

Draft files:

- `intro.md`
- `related_work.md`
- `method.md`
- `experiments.md`
- `results_tables.md`
- `limitations.md`
- `figures.md`

Control file:

- `handoff.md`: update only for Tier A/B direction, evidence, or writing milestones.

Local storage:

- `papers/`: optional local PDFs, ignored by default.
- `notes/`: reading notes for important papers.

Rules:

- Literature collection does not automatically trigger paper prose.
- Results are table templates and TODO placeholders unless actual numbers are provided.
- Claims must be traceable to repo evidence, user-provided results, cited literature, user decisions, or explicit assumptions.
- Venue/outlet profile records writing tendencies only; it is not a replacement for human writing judgment.
